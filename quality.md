# 质量规范与检查清单

本文件详细说明文案生成的质量约束、检查清单和语言规范。

---

## 核心约束（MUST 遵守）

### CRITICAL 规则

1. **MUST** - 所有产品描述必须基于图片可见信息
2. **NEVER** - 禁止虚构产品功能、效果、材质
3. **NEVER** - 禁止使用"匠心打造""品质之选""极致体验"等空洞词汇

### 输出格式约束

4. **MUST** - 严格按照对应平台的模板结构输出
5. **MUST** - 每个版本必须附带【字数统计】【风格标签】
6. **MUST** - 2-3个版本之间必须在侧重点/角度/结尾策略上有明显差异

### 质量红线

7. **NEVER** - 禁止编造用户评价、销量数据
8. **NEVER** - 禁止虚假对比（"比XX品牌好"）
9. **NEVER** - 禁止连续使用超过2个四字形容词

---

## 真实性准则

1. **只描述图片可见内容** - 不虚构产品特性
2. **不夸大产品功效** - 避免过度承诺
3. **材质描述需有依据** - 不凭空猜测
4. **场景描述基于图片** - 不编造使用场景
5. **功能描述需谨慎** - 只陈述可确认的功能

---

## 语言自然度要求

### 避免词汇替换表

| 要避免 | 建议替换 | 原因 |
|--------|----------|------|
| "这款产品具有..." | "这个..." / "它..." | 过于正式，像说明书 |
| "让您的生活更加美好" | 具体描述带来的改变 | 空洞无物 |
| "匠心打造，品质之选" | 具体的工艺或品质描述 | 过度包装 |
| "完美呈现..." | "呈现..." / "展现..." | 夸大其词 |
| "极致体验" | 具体描述体验内容 | 无法验证 |
| "为您带来极致体验" | 具体描述体验内容 | 过度承诺 |
| 过多形容词堆砌 | 选择1-2个核心形容词 | 显得浮夸 |
| 生硬的排比句 | 自然流畅的表达 | AI感重 |
| 连续超过2个四字形容词 | 拆开或替换为口语化表达 | 过于书面化 |

### 空洞词汇黑名单

以下词汇在文案中**禁止使用**：

- 匠心打造
- 品质之选
- 极致体验
- 完美呈现
- 尊贵享受
- 奢华体验
- 顶级品质
- 无与伦比的
- 颠覆性的
- 革命性的

## 禁止事项

- 禁止虚构产品功能或效果
- 禁止夸大材质或工艺
- 禁止使用无法验证的承诺
- 禁止过度营销化表达
- 禁止抄袭他人文案
- 禁止使用虚假对比（"比XX品牌好"）
- 禁止编造用户评价、销量数据
- 禁止连续使用超过2个四字形容词
- 禁止使用"匠心打造""品质之选""极致体验"等空洞词汇

## 质量检查清单

生成文案后，需逐项自查：

### 真实性检查
- [ ] 所有描述是否基于图片真实信息？
- [ ] 是否存在夸大或虚构内容？
- [ ] 是否编造了用户评价或销量数据？
- [ ] 是否存在虚假对比？

### 语言质量检查
- [ ] 是否使用了"匠心打造""品质之选"等空洞词汇？
- [ ] 是否连续使用了超过2个四字形容词？
- [ ] 语言是否自然流畅，无AI生硬感？
- [ ] 是否有错别字或语病？

### 格式规范检查
- [ ] 是否符合目标平台调性？
- [ ] 是否符合用户指定的风格？
- [ ] 字数是否在要求范围内？
- [ ] 每个版本是否附带【字数统计】【风格标签】？
- [ ] 多版本之间是否有明显差异（侧重点/角度/结尾策略）？
- [ ] 结尾是否自然收束？

---

## 版本差异策略检查

确保2-3个版本之间有明显差异：

| 检查项 | 版本一 | 版本二 | 版本三 |
|--------|--------|--------|--------|
| 侧重点 | 情感共鸣 | 品质展示 | 转化促单 |
| 开头策略 | 个人感受引入 | 产品定位点明 | 直接推荐 |
| 主体内容 | 使用体验+心情 | 工艺细节+设计理念 | 核心卖点+性价比 |
| 结尾策略 | 互动引导 | 品牌调性升华 | 明确行动引导 |

---

## 质量评分标准

| 等级 | 标准 | 说明 |
|------|------|------|
| A级 | 全部检查项通过 | 可直接使用 |
| B级 | 1-2项轻微问题 | 微调后使用 |
| C级 | 3-5项问题 | 需要修改 |
| D级 | 5项以上问题或严重违规 | 重新生成 |

**降级处理：**
- 如生成文案被评为C级或D级，需重新生成并记录问题原因
- 连续3次生成不达标，建议引导用户手动输入补充信息
