# 质量规范与检查清单

本文件详细说明文案生成的质量约束、检查清单和语言规范。

---

## 核心约束（MUST 遵守）

### CRITICAL 规则

1. **MUST** - 所有产品描述必须基于图片可见信息
2. **NEVER** - 禁止虚构产品功能、效果、材质
3. **NEVER** - 禁止使用"匠心打造""品质之选""极致体验"等空洞词汇

### 输出格式约束

4. **MUST** - 严格按照对应平台的模板结构输出
5. **MUST** - 每个版本必须附带【字数统计】【风格标签】
6. **MUST** - 2-3个版本之间必须在侧重点/角度/结尾策略上有明显差异

### 质量红线

7. **NEVER** - 禁止编造用户评价、销量数据
8. **NEVER** - 禁止虚假对比（"比XX品牌好"）
9. **NEVER** - 禁止连续使用超过2个四字形容词

---

## 真实性准则

1. **只描述图片可见内容** - 不虚构产品特性
2. **不夸大产品功效** - 避免过度承诺
3. **材质描述需有依据** - 不凭空猜测
4. **场景描述基于图片** - 不编造使用场景
5. **功能描述需谨慎** - 只陈述可确认的功能

---

## 语言自然度要求

### 避免词汇替换表

| 要避免 | 建议替换 | 原因 |
|--------|----------|------|
| "这款产品具有..." | "这个..." / "它..." | 过于正式，像说明书 |
| "让您的生活更加美好" | 具体描述带来的改变 | 空洞无物 |
| "匠心打造，品质之选" | 具体的工艺或品质描述 | 过度包装 |
| "完美呈现..." | "呈现..." / "展现..." | 夸大其词 |
| "极致体验" | 具体描述体验内容 | 无法验证 |
| "为您带来极致体验" | 具体描述体验内容 | 过度承诺 |
| 过多形容词堆砌 | 选择1-2个核心形容词 | 显得浮夸 |
| 生硬的排比句 | 自然流畅的表达 | AI感重 |
| 连续超过2个四字形容词 | 拆开或替换为口语化表达 | 过于书面化 |

### 空洞词汇黑名单

以下词汇在文案中**禁止使用**：

- 匠心打造
- 品质之选
- 极致体验
- 完美呈现
- 尊贵享受
- 奢华体验
- 顶级品质
- 无与伦比的
- 颠覆性的
- 革命性的

## 禁止事项

- 禁止虚构产品功能或效果
- 禁止夸大材质或工艺
- 禁止使用无法验证的承诺
- 禁止过度营销化表达
- 禁止抄袭他人文案
- 禁止使用虚假对比（"比XX品牌好"）
- 禁止编造用户评价、销量数据
- 禁止连续使用超过2个四字形容词
- 禁止使用"匠心打造""品质之选""极致体验"等空洞词汇

## 质量检查清单

生成文案后，需逐项自查：

### 真实性检查
- [ ] 所有描述是否基于图片真实信息？
- [ ] 是否存在夸大或虚构内容？
- [ ] 是否编造了用户评价或销量数据？
- [ ] 是否存在虚假对比？

### 语言质量检查
- [ ] 是否使用了"匠心打造""品质之选"等空洞词汇？
- [ ] 是否连续使用了超过2个四字形容词？
- [ ] 语言是否自然流畅，无AI生硬感？
- [ ] 是否有错别字或语病？

### 格式规范检查
- [ ] 是否符合目标平台调性？
- [ ] 是否符合用户指定的风格？
- [ ] 字数是否在要求范围内？
- [ ] 每个版本是否附带【字数统计】【风格标签】？
- [ ] 多版本之间是否有明显差异（侧重点/角度/结尾策略）？
- [ ] 结尾是否自然收束？

---

## 版本差异策略检查

确保2-3个版本之间有明显差异：

| 检查项 | 版本一 | 版本二 | 版本三 |
|--------|--------|--------|--------|
| 侧重点 | 情感共鸣 | 品质展示 | 转化促单 |
| 开头策略 | 个人感受引入 | 产品定位点明 | 直接推荐 |
| 主体内容 | 使用体验+心情 | 工艺细节+设计理念 | 核心卖点+性价比 |
| 结尾策略 | 互动引导 | 品牌调性升华 | 明确行动引导 |

## 去AI味写作核心技巧

### 为什么AI写的文案有"AI味"

AI文案的典型特征（必须避免）：

| AI味特征 | 具体表现 | 为什么像AI |
|----------|----------|------------|
| 结构太工整 | 每段字数差不多，总分总结构明显 | 真人写作不会这么均匀 |
| 排比太多 | "不仅...而且...更..."连续排比 | AI喜欢用排比凑字数 |
| 情感空洞 | "真的被圈粉了""狠狠拿捏了" | 网络用语堆砌，缺乏真实感受 |
| 信息密度低 | 一句话能说清楚的事用三句 | AI倾向于"解释"而不是"呈现" |
| 过渡词生硬 | "此外""不仅如此""值得一提的是" | 书面连接词，口语中不会用 |
| 结尾套路 | "真心推荐""赶紧冲""值得入手" | 每篇都用一样的收尾 |
| 标点单一 | 全是逗号和句号，没有语气变化 | 缺少真人写作的节奏感 |

### 写像真人的6个技巧

**技巧1：用"不完美"的句子结构**

真人写作会有长短句交替、口语化表达、偶尔的跑题：

```
❌ AI味：
这个杯子的外观设计非常简约，奶白色的色调让人感到温馨，
金色的描边增添了精致感，放在桌面上十分好看。

✅ 像真人：
奶白色，很温柔的那种白，不是冷冰冰的白。
杯口一圈金色描边，细细的一条，不夸张但就是好看。
我把它放在电脑旁边，打字累了抬头看一眼，心情会好一点。
```

**技巧2：加入"私人细节"**

真人分享时会带个人生活细节，而不是纯产品描述：

```
❌ AI味：
这款陶瓷杯适合居家办公使用，搭配木质桌面效果更佳。

✅ 像真人：
我平时在家办公，桌子不大，东西不敢多放。
但这个杯子往那一搁，整个桌面好像就活了。
旁边什么都不用摆，它自己就是装饰。
```

**技巧3：用"具体数字+对比"替代形容词**

```
❌ AI味：
拿在手里很有分量，质感非常好。

✅ 像真人：
比想象中重一点，大概跟一个苹果差不多。
之前用的那个塑料杯轻飘飘的，换成这个之后，
喝水的仪式感一下就不一样了。
```

**技巧4：适当"跑题"再拉回来**

真人写作不会一直说产品，会穿插一些无关但有趣的内容：

```
❌ AI味：
这款陶瓷杯采用优质陶瓷材质，哑光质感，触感细腻。

✅ 像真人：
说实话我之前一直用公司发的马克杯，将就用了一年多。
也不是不好，就是每次喝水都没什么感觉，像完成任务。
直到上周同事推荐了这个，我才意识到原来杯子也能让人开心。
陶瓷的，哑光的，摸上去有点像鹅卵石那种滑滑的感觉。
```

**技巧5：结尾不要总结，要"留白"**

```
❌ AI味：
综上所述，这款陶瓷杯无论是外观还是质感都非常出色，
真心推荐给大家，值得入手！

✅ 像真人：
反正我现在每天早上第一件事就是用它泡咖啡。
以前没觉得杯子有什么重要的。
```

**技巧6：善用短句和断句，制造节奏感**

```
❌ AI味：
奶白色的陶瓷杯身搭配金色描边设计，整体呈现出简约而不简单的视觉效果。

✅ 像真人：
奶白色。金色描边。就这两个元素。
但放在一起，就是好看。
说不上来为什么，可能越简单的东西越耐看吧。
```

### AI味自检清单

生成文案后，逐条检查：

- [ ] 是否每段字数差不多？（真人写作段落长短不一）
- [ ] 是否有3个以上的排比句？（真人很少连续排比）
- [ ] 是否用了"此外""不仅如此""值得一提的是"？（口语中不会用）
- [ ] 结尾是否在"总结推荐"？（真人往往不总结，直接结束）
- [ ] 是否通篇都在说产品？（真人会穿插个人生活）
- [ ] 是否每句话都很"正确"但很"空"？（真人写作有主观感受）
- [ ] 标点是否只有逗号和句号？（真人会用省略号、破折号、感叹号）
- [ ] 读出来是否像在"念稿"？（真人写作像在"说话"）

## 质量评分标准

| 等级 | 标准 | 说明 |
|------|------|------|
| A级 | 全部检查项通过 | 可直接使用 |
| B级 | 1-2项轻微问题 | 微调后使用 |
| C级 | 3-5项问题 | 需要修改 |
| D级 | 5项以上问题或严重违规 | 重新生成 |

**降级处理：**
- 如生成文案被评为C级或D级，需重新生成并记录问题原因
- 连续3次生成不达标，建议引导用户手动输入补充信息
