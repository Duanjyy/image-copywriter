# 文案模板库

本文件提供各平台的文案模板和示例，供生成文案时参考。

---

## 一、朋友圈文案模板

### 温柔种草风格

**模板结构：**
```
[产品感受开篇] + [1-2个细节亮点] + [使用场景/心情] + [互动收尾]
```

**示例：**
```
今天收到的小惊喜，打开就被这个质感圈粉了。
奶白色的陶瓷杯身，拿在手里刚刚好的重量感，
杯口的金色描边特别精致。
放在办公桌上，每次喝水都是一种小确幸。
你们觉得好看吗？
```

### 简约高级风格

**模板结构：**
```
[产品定位] + [核心卖点] + [品质感收尾]
```

**示例：**
```
极简设计，质感呈现。
陶瓷哑光质感，金色描边点缀细节。
于日常中，遇见美好。
```

### 接地气带货风格

**模板结构：**
```
[直接推荐] + [核心卖点] + [促单引导]
```

**示例：**
```
这个真的要推荐给你们！
陶瓷杯，质感超好，奶白色百搭又好看。
我自己都囤了两个，趁活动赶紧冲！
```

## 三、电商详情文案模板

**模板结构：**
```
[产品名称/定位]

[核心卖点概述]

【产品亮点】
- 亮点一：具体描述
- 亮点二：具体描述
- 亮点三：具体描述

【材质工艺】
材质说明与工艺特点

【使用场景】
场景描述与搭配建议

【品质承诺】
品质保障说明
```

**示例：**
```
极简陶瓷马克杯 | 品质生活之选

一款让日常喝水都变得美好的杯子。

【产品亮点】
- 奶白色调：温柔百搭，适配各种家居风格
- 金色描边：精致细节，提升整体质感
- 舒适手柄：弧度设计，握感舒适

【材质工艺】
精选优质陶瓷，经过高温烧制而成。
哑光质感表面，触感细腻温润。
杯口金色描边工艺，经久耐用不易脱落。

【使用场景】
适合居家办公、下午茶时光、书房阅读等场景。
可搭配木质桌面、绿植、书籍等元素，
营造温馨舒适的生活氛围。

【品质承诺】
品质保证，如有质量问题可退换。
```

## 九、Amazon文案模板

**模板结构：**
```
Title: [核心关键词+规格+卖点]

Bullet Points:
• [卖点1]
• [卖点2]
• [卖点3]
• [卖点4]
• [卖点5]

Description:
[场景化描述+品牌故事+使用体验]
```

**示例：**
```
Title: Ceramic Coffee Mug 12oz, Cream White with Gold Rim, Dishwasher Safe

Bullet Points:
• Elegant cream-white ceramic with hand-painted gold rim adds sophistication
• Generous 12oz capacity perfect for coffee, tea, hot chocolate
• Microwave and dishwasher safe for convenient everyday use
• Comfortable ergonomic handle designed for secure grip
• Makes an ideal gift for housewarmings, weddings, birthdays

Description:
Elevate your daily coffee ritual with this beautifully crafted ceramic mug. 
The cream-white glaze paired with delicate gold trim creates a timeless aesthetic. 
Made from high-quality ceramic, this mug is both durable and elegant.
```
