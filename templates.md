# 文案模板库

本文件提供各平台的文案模板和示例，供生成文案时参考。

**CRITICAL：所有示例均为"真人写作风格"参考，生成文案时必须模仿这种自然、有温度、有私人细节的写法，禁止写成工整排比的AI文体。**

---

## 写作篇幅要求

| 平台 | 最低字数 | 建议字数 | 篇幅要求 |
|------|----------|----------|----------|
| 朋友圈 | 80字 | 100-200字 | 不要太短，3-5段，有起承转合 |
| 小红书 | 300字 | 400-600字 | 至少5段，有细节有感受有场景 |
| 电商详情 | 400字 | 500-800字 | 信息完整但不啰嗦 |
| 私域带货 | 150字 | 200-350字 | 像跟朋友聊天，自然展开 |
| 抖音/快手 | 80字 | 100-200字 | 短但有信息量 |
| 微博 | 120字 | 150-300字 | 有观点有互动 |
| B站 | 300字 | 400-700字 | 像UP主在跟粉丝聊天 |
| Instagram | 60字 | 80-200字 | 简洁但不空洞 |

**MUST：宁可写得稍长一些有内容，也不要写得很短像广告语。**

## 一、朋友圈文案模板

### 温柔种草风格

**模板结构：**
```
[个人生活引入/心情] + [产品细节描述，带私人感受] + [使用场景，带生活细节] + [自然收尾，不总结]
```

**示例（约150字）：**
```
最近加班多，桌上的东西越堆越多，但这个杯子我一直留着最好的位置。

奶白色，不是那种冷冷的白，是带点暖调的奶白。杯口一圈金色描边，很细，不仔细看差点没注意到。但就是这种不张扬的细节，越看越喜欢。

拿起来比想象中重一点，大概跟一个苹果差不多吧。之前用的塑料杯轻飘飘的，喝水像完成任务，换了它之后莫名其妙愿意多喝几杯。

也不是什么贵东西，但每天早上用它泡第一杯咖啡的时候，觉得这一天好像也没那么难熬。
```

### 简约高级风格

**模板结构：**
```
[简洁的产品定位] + [1-2个核心细节，用短句呈现] + [留白式收尾]
```

**示例（约100字）：**
```
奶白色。金色描边。就这两个元素。

但放在一起，就是好看。说不上来为什么，可能越简单的东西越耐看吧。

哑光的陶瓷表面，摸上去有点像鹅卵石那种滑滑的触感。不是光滑，是温润。

放在桌上不抢眼，但你会一直注意到它。
```

### 接地气带货风格

**模板结构：**
```
[直接推荐+原因] + [产品细节+使用感受] + [价格/优惠信息] + [促单引导]
```

**示例（约160字）：**
```
姐妹们这个杯子我真的要推，不是广告不是赞助，纯自己花钱买的。

之前一直用公司发的马克杯，将就用了一年多。上周实在看不下去了，挑了好久选了这款。奶白色，金色描边，颜值没得说。关键是拿在手里有分量，不是那种轻飘飘的廉价感。

我自己用了快两周了，每天早上泡咖啡都用它。同事来我工位看到了也问我要链接。

现在好像还在做活动，比我买的时候便宜了十几块。想要的趁现在。
```

## 二、小红书文案模板

### 温柔种草风格

**模板结构：**
```
[吸引人的标题，不要太标题党]
[个人生活背景引入，建立共鸣]
[产品细节描述，穿插私人感受和对比]
[使用场景描述，带生活细节]
[自然收尾，不总结不推荐]
```

**示例（约450字）：**
```
标题：搬进新家后买的最让我开心的小东西

上个月搬了家，东西还没收拾完的时候，我先下单了一个杯子。

听起来有点离谱对吧，家具都没买齐先买杯子。但就是那种感觉，新家新开始，想从一个每天都会用到的小物件开始。

挑了好久，最后选了这款奶白色的陶瓷杯。收到的时候包装很简单，一个纸盒，没有花里胡哨的设计。打开看到实物的第一反应是——嗯，就是这个感觉。

奶白色，很温柔的那种白。不是冷冰冰的瓷白，是带一点点暖调的，像加了牛奶进去的那种颜色。杯口有一圈金色描边，很细很细的一条线，不夸张，不张扬，但你拿近了看会觉得好精致。

拿起来比想象中重。我之前用的那个宜家的玻璃杯，轻飘飘的，喝水没什么感觉。这个拿在手里沉甸甸的，莫名让人觉得喝进去的水都变好喝了（虽然知道是心理作用）。

我现在每天早上用它泡咖啡，放在书桌靠窗的位置。阳光照进来的时候，杯口的金色描边会反光，一闪一闪的。

也不是什么贵东西，但就是那种让你每天看到都会心情变好的物件。搬家的时候买了很多东西，大部分都记不清了，但这个杯子是我第一个想推荐的。
```

### 简约高级风格

**模板结构：**
```
[简洁标题]
[产品定位，一两句话]
[设计细节，用短句和断句呈现]
[使用感受，点到为止]
[留白收尾]
```

**示例（约300字）：**
```
标题：一个杯子能有多好看

奶白色。哑光釉面。金色描边。

就这三个元素，没有第四个了。

杯身是圆润的弧线，不是那种棱角分明的设计，握在手里刚好贴合手掌的弧度。哑光的表面不像亮面陶瓷那样反光，而是那种柔和的漫反射，看着很舒服。

杯口的金色描边是手绘的，仔细看会有一点点不规则。我反而觉得这个挺好的，太完美了反而像机器生产的。

350ml，不大不小。泡一杯美式刚好，泡茶也行。

我买的时候没想那么多，就是觉得好看。用了一段时间之后发现，好的设计就是这样——你不会每天注意到它，但拿走之后会觉得少了点什么。
```

### 接地气带货风格

**模板结构：**
```
[标题：直接说产品+价格/优惠]
[个人背景+为什么买]
[产品细节+真实使用感受]
[对比之前用的产品]
[价格信息+促单引导]
```

**示例（约400字）：**
```
标题：不到50块的杯子，质感吊打我之前100多的

先说结论：这个杯子是我今年买过最值的小东西，没有之一。

起因是我之前那个杯子摔了。用了快两年的马克杯，杯盖的塑料裂了，喝热水的时候会漏。本来想随便买个新的凑合用，结果刷到这款，看颜值就下单了。

收到之后说实话有点意外。这个价位我没指望质感多好，但拿到手确实比想象中好。陶瓷的，有一定重量，不是那种轻飘飘的。表面是哑光的，摸着滑滑的但不打滑，像鹅卵石那种触感。

奶白色配金色描边，放桌上挺好看的。我同事路过我工位说了句"这杯子不错啊"，还以为我花了一百多。

容量350ml，泡咖啡泡茶都够用。我试了一下放微波炉加热，没问题。洗碗机也洗过一次，金色描边没有掉。

现在好像做活动，比我买的时候还便宜了5块钱。这个价格说实话闭眼入，比超市里那些二三十的杯子质感好太多了。
```

## 三、电商详情文案模板

**模板结构：**
```
[产品名称/一句话定位]

[使用场景描述，带入感强]

【产品细节】
- 细节一：具体描述+使用感受
- 细节二：具体描述+使用感受
- 细节三：具体描述+使用感受

【材质与工艺】
用通俗的语言描述，不要像说明书

【适合谁用】
具体场景，具体人群

【规格参数】
客观信息，简洁呈现
```

**示例（约500字）：**
```
极简陶瓷马克杯 — 那种让你每天早上都想用的杯子

你有没有过这种感觉：早上起床，迷迷糊糊去泡咖啡，打开橱柜看到一排杯子，但就是没有一个让你"就它了"的。

我之前就是这样。直到换了这款。

【它为什么不一样】

奶白色的杯身不是那种冷白的瓷色，而是带暖调的奶白，像往白色里加了一滴牛奶。放在厨房的台面上，跟木质砧板、白色瓷砖都很搭，不会突兀。

杯口的金色描边是手绘的，仔细看会有一丝丝不规则。这个不是瑕疵，是手工的痕迹。比机器印上去的那种死板的金线有温度多了。

哑光的釉面摸上去很舒服。不是那种光滑到反光的亮面，而是像鹅卵石一样的柔和触感。手柄的弧度刚好，四根手指穿过去不会觉得挤，大拇指自然搭在杯身上，握感很稳。

【关于材质】

高温烧制的陶瓷，不是低温釉那种用久了会掉色的。可以放微波炉加热，也可以丢洗碗机。金色描边用的是真金水，正常使用不会脱落。

【适合这些场景】

- 早上泡咖啡或茶，开启一天
- 放在办公桌上，工作时随手喝
- 下午茶时间，配一块饼干
- 送朋友搬家或入职的小礼物

【规格】
容量：350ml | 材质：高温陶瓷 | 颜色：奶白色+金色描边
适用：微波炉✓ 洗碗机✓
```
