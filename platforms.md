# 平台适配指南

本文件详细说明10个发布平台的特性、调性和文案要求。

---

## 平台特性对比

| 平台 | 最低字数 | 建议字数 | 内容特点 | 核心诉求 |
|------|----------|----------|----------|----------|
| 朋友圈 | 80字 | 100-200字 | 简洁有温度，生活化 | 引发共鸣，促进互动 |
| 小红书 | 300字 | 400-600字 | 种草感强，细节丰富 | 激发购买欲，分享价值 |
| 电商详情 | 400字 | 500-800字 | 信息完整，卖点清晰 | 转化导向，建立信任 |
| 私域带货 | 150字 | 200-350字 | 亲切直接，促单明确 | 快速成交，关系维护 |
| 抖音/快手短视频 | 80字 | 100-200字 | 口语化、节奏快、前3秒抓人 | 停留观看，引导下单 |
| 微博 | 120字 | 150-300字 | 热点感、话题性强 | 传播扩散，引发讨论 |
| B站 | 300字 | 400-700字 | 专业感、有趣有料、UP主口吻 | 种草安利，弹幕互动 |
| Instagram/海外平台 | 60字 | 80-200字 | 视觉导向、简洁有力、hashtag | 品牌调性，粉丝互动 |
| Amazon/跨境电商 | 300字 | 400-1000字 | SEO导向、功能描述、结构化 | 搜索排名，转化购买 |
| eBay/拍卖电商 | 200字 | 250-500字 | 信任导向、个人化、详细规格 | 建立信任，促成拍卖 |

> **篇幅原则**：宁可写得稍长有内容，也不要写得太短像广告语。真人写作天然有一定的篇幅，因为会穿插个人感受、生活细节和场景描述。

## 平台调性指南

### 朋友圈

- 像朋友分享，不像广告推销
- 可适当加入个人感受
- 结尾可带互动引导
- 避免过度营销化表达
- 语言口语化，有生活气息

### 小红书

- 标题要有吸引力（可适当使用emoji点缀）
- 正文分段清晰，善用空行
- 可融入使用体验或场景描述
- 种草感要自然，不刻意
- 可带话题标签

### 电商详情

- 结构化呈现：卖点→细节→场景
- 信息准确完整
- 可适当强调品质感
- 突出产品差异化优势
- 建立信任感

### 私域带货

- 口语化，像聊天推荐
- 可带限时/限量信息
- 明确行动引导
- 突出性价比
- 维护客户关系

### 抖音/快手短视频

- 前3秒必须抓住注意力（痛点/悬念/反差）
- 口语化表达，像跟朋友聊天
- 节奏快，不啰嗦，每句话都有信息量
- 结尾明确引导：点赞/关注/下单
- 可配合视频画面节奏分段

**短视频文案模板：**
```
[前3秒钩子：痛点/悬念/反差] + [产品展示+核心卖点] + [使用场景/效果] + [促单引导]
```

**示例：**
```
每次喝水都嫌杯子太丑？（钩子）
看看这个，奶白色陶瓷杯，金色描边，
放在桌上就是一道风景。（产品展示）
关键是拿在手里沉甸甸的，质感绝了。（核心卖点）
链接放下方，冲就完了。（促单引导）
```

### 微博

- 开头要有话题感或新闻感
- 可带热搜话题标签 #xxx#
- 语言简洁有力，适合快速浏览
- 可适当加入观点或态度
- 结尾引导转发/评论

**微博文案模板：**
```
[话题引入/观点] + [产品亮点] + [互动引导]
```

**示例：**
```
#好物推荐# 最近入手了一个超好看的陶瓷杯，奶白色配金色描边，放在办公桌上整个角落都温柔了。有同款吗？评论区聊聊～
```

### B站

- 用UP主的口吻，像在跟粉丝聊天
- 可以稍微"整活"，有趣有梗
- 专业感和亲和力并存
- 可带弹幕互动引导（"弹幕打出xxx"）
- 适合做开箱/测评类文案

**B站文案模板：**
```
[UP主开场白] + [产品开箱/展示] + [详细体验/测评] + [总结推荐]
```

**示例：**
```
大家好，今天给大家开箱一个我期待很久的杯子。
奶白色陶瓷质感，拿在手里很有分量，
杯口的金色描边是点睛之笔。
说实话这个质感对得起这个价格，
放在桌上是真的好看。想要的同学链接放简介了。
```

### Instagram / 海外平台

- 视觉导向，文案简洁有力
- 善用 emoji 和 hashtag
- 英文表达自然地道，不生硬翻译
- 可带 CTA（Call to Action）
- 注重品牌调性和美感

**Instagram 文案模板：**
```
[简洁有力的开头] + [产品亮点] + [hashtag] + [CTA]
```

**示例：**
```
Simple elegance for your everyday ☕
This cream-white ceramic mug with gold trim is everything.
Minimalist design, maximum vibes.

Shop via link in bio 🔗

#ceramicmug #minimalstyle #desksetup #aestheticfinds #coffeelover
```

### Amazon / 跨境电商

- SEO导向，关键词前置
- 结构化呈现：标题+5点卖点+描述
- 突出功能、规格、使用场景
- 避免主观形容词，强调事实
- 符合 Amazon A9 算法优化

**Amazon 文案模板：**
```
[产品标题：核心关键词+规格+卖点]

[5点卖点：每点一个核心功能/优势]

[产品描述：场景化描述+品牌故事+使用体验]
```

**示例：**
```

• Generous 12oz capacity perfect for coffee, tea, hot chocolate, or your favorite beverage

```

### eBay / 拍卖电商

- 标题关键词堆砌（80字符限制）
- 突出性价比和促销信息
- 详细的物品规格和状态说明
- 强调物流和退换政策
- 个人化、可信的卖家口吻

**eBay 文案模板：**
```
[标题：关键词堆砌+品牌+规格+状态]

[物品描述：详细规格+状态+使用体验]

[卖家信息：可信度+服务承诺]
```

## 平台选择决策树

不确定选哪个平台？参考以下决策树：

```
你的产品适合发在哪里？
├── 日常分享，朋友可见 → 朋友圈
├── 种草安利，图文详细 → 小红书
├── 视频带货，节奏快 → 抖音/快手
├── 热点话题，传播扩散 → 微博
├── 专业测评，有趣有料 → B站
├── 海外用户，英文输出 → Instagram
├── 电商卖货，转化导向 → 电商详情/Amazon/eBay
└── 私域成交，关系维护 → 私域带货
```
