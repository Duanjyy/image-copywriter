# Image Copywriter - 图片文案生成器

上传一张产品图，自动生成朋友圈、小红书、电商详情、私域带货等多平台文案。

---

## 功能特点

- **智能图片识别** - 自动提取产品外观、材质、细节、场景信息
- **多平台适配** - 支持朋友圈、小红书、电商详情、私域带货四大场景
- **多种风格可选** - 温柔种草、简约高级、接地气带货三种风格
- **真实性保障** - 严格依据图片真实信息创作，不虚构不夸大
- **自然语言输出** - 避免AI生硬感，文案像人写的

---

## 使用方法

### 1. 上传图片

将产品图片上传给 SOLO，支持 PNG、JPG、JPEG、GIF、WEBP 格式。

### 2. 确认识别结果

系统会自动识别图片中的产品信息：
- 产品外观（形状、颜色、设计特点）
- 材质质感（材质类型、触感、光泽）
- 细节亮点（工艺细节、品牌元素）
- 场景氛围（使用场景、搭配元素）

确认信息准确后进入下一步。

### 3. 选择发布平台

- **朋友圈** - 50-150字，简洁有温度
- **小红书** - 200-500字，种草感强
- **电商详情** - 300-800字，转化导向
- **私域带货** - 100-300字，促单明确

### 4. 选择文案风格

- **温柔种草** - 细腻柔和，情感共鸣
- **简约高级** - 精炼克制，质感导向
- **接地气带货** - 直白亲切，促单明确

### 5. 获取文案

系统生成 2-3 个版本的文案供选择，可直接使用或微调。

## 示例

**输入图片：** 奶白色陶瓷杯，金色描边，置于木质桌面

**识别结果：**
```
外观：圆润杯身，奶白色调
材质：陶瓷，哑光质感
细节：杯口金色描边
场景：木质桌面，温馨居家感
```

**输出文案（小红书/温柔种草）：**

```
标题：被问爆了的宝藏好物

真的被这个杯子狠狠拿捏了！

奶白色的陶瓷质感，拿在手里沉甸甸的很有分量。
杯口的金色描边是点睛之笔，精致又不张扬。
放在书桌上，搭配一束小花，整个角落都温柔起来了。

每天用它喝水，心情都会变好。
真心推荐给喜欢极简风的朋友们～
```

## 核心原则

1. **只写真实存在的东西** - 不虚构产品特性
2. **说人话** - 不用"匠心打造""品质之选"等空洞词汇
3. **适配场景** - 不同平台用不同写法
4. **多版本可选** - 提供选择，而非唯一答案
