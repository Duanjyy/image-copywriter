# Image Copywriter - 图片文案生成器

上传一张产品图，自动生成朋友圈、小红书、电商详情、私域带货等**10个平台**的文案。

**最后更新：2026.5.20**

---

## 功能特点

- **智能图片识别** - 自动提取产品外观、材质、细节、场景信息
- **10平台适配** - 朋友圈、小红书、抖音/快手、微博、B站、电商详情、私域带货、Instagram、Amazon、eBay
- **双模式选择** - 快速模式2步完成 / 完整模式7步定制
- **品牌层级选择** - 大众/中端/高端/奢侈 4级品牌调性
- **多种风格可选** - 温柔种草、简约高级、接地气带货
- **多图组合处理** - 智能识别主图、细节图、场景图、对比图
- **中英双语输出** - 支持中文、英文、中英双语
- **真实性保障** - 严格依据图片真实信息创作，不虚构不夸大
- **版本选择指南** - 测点击率/转化率/品牌调性的智能推荐

## 使用方法

### 1. 触发 Skill

说出唤醒语：
- "帮我图片文案生成"
- "我要写小红书文案"
- "产品文案创作"
- "image copywriting"
- "配个文案" / "帮图写段话" / "种草文"

### 2. 选择模式

**快速模式（推荐）**：
- 只需选择平台和风格，2步完成
- 品牌层级、语言等自动智能推荐
- 适合追求效率的用户

**完整模式**：
- 自定义所有参数，7步完成
- 品牌层级、字数、语言等全部可选
- 适合精细化定制需求

### 3. 上传图片

将产品图片上传给 SOLO，支持 PNG、JPG、JPEG、GIF、WEBP 格式。

支持单图和多图组合（2-3张/4张+）。

### 4. 确认参数

根据选择的模式，确认以下参数：
- 发布平台（10选1）
- 品牌层级（大众/中端/高端/奢侈）
- 文案风格（温柔种草/简约高级/接地气带货）
- 字数要求（按平台默认/自定义）
- 输出语言（中文/英文/双语）

### 5. 获取文案

系统生成 2-3 个版本的文案供选择，附带：
- 【字数统计】
- 【风格标签】
- 【版本定位】（情感共鸣型/品质导向型/促单转化型）

并提供版本选择建议（测点击率/转化率/品牌调性）。

## 示例

**输入图片：** 奶白色陶瓷杯，金色描边，置于木质桌面

**识别结果：**
```
外观：圆润杯身，奶白色调
材质：陶瓷，哑光质感
细节：杯口金色描边
场景：木质桌面，温馨居家感
```

**输出文案（小红书/温柔种草）：**

```
标题：被问爆了的宝藏好物

真的被这个杯子狠狠拿捏了！

奶白色的陶瓷质感，拿在手里沉甸甸的很有分量。
杯口的金色描边是点睛之笔，精致又不张扬。
放在书桌上，搭配一束小花，整个角落都温柔起来了。

每天用它喝水，心情都会变好。
真心推荐给喜欢极简风的朋友们～
```

**输出文案（Instagram/英文）：**

```
Simple elegance for your everyday ☕
This cream-white ceramic mug with gold trim is everything.
Minimalist design, maximum vibes.

Shop via link in bio 🔗

#ceramicmug #minimalstyle #desksetup #aestheticfinds
```

## 核心原则

1. **只写真实存在的东西** - 不虚构产品特性
2. **说人话** - 不用"匠心打造""品质之选"等空洞词汇
3. **适配场景** - 不同平台用不同写法
4. **多版本可选** - 提供选择，而非唯一答案
5. **品牌调性匹配** - 大众/中端/高端/奢侈 4级精准匹配
