# Image Copywriter - 图片文案生成器

上传一张产品图，自动生成朋友圈、小红书、抖音、微博、B站、Instagram 等8大平台文案。支持多图组合、品牌层级选择、中英双语输出。

> 最后更新：2026年5月20日

---

## 功能特点

- **智能图片识别** - 自动提取产品外观、材质、细节、场景、图中文字信息
- **8大平台适配** - 朋友圈、小红书、电商详情、私域带货、抖音/快手、微博、B站、Instagram
- **品牌层级选择** - 大众/中端/高端/奢侈，四级品牌调性精准匹配
- **多种风格可选** - 温柔种草、简约高级、接地气带货三种风格
- **多图组合模式** - 支持多张图片组合识别，主图+细节+场景串联成文
- **中英双语输出** - 国内平台中文，海外平台英文，支持双语模式
- **图片质量检测** - 模糊/主体不明自动提醒，含人脸隐私保护
- **降级策略** - 识别不足时尽力生成并标注，完全失败可手动输入
- **真实性保障** - 严格依据图片真实信息创作，不虚构不夸大
- **自然语言输出** - 避免AI生硬感，文案像人写的

## 使用方法

### 1. 上传图片

将产品图片上传给 SOLO，支持单图或多图，格式：PNG、JPG、JPEG、GIF、WEBP。

### 2. 确认识别结果

系统会自动识别图片中的产品信息：
- 产品外观（形状、颜色、设计特点）
- 材质质感（材质类型、触感、光泽）
- 细节亮点（工艺细节、品牌元素）
- 场景氛围（使用场景、搭配元素）
- 图中文字（品名、成分、价格、规格等）

确认信息准确后进入下一步。

### 3. 选择发布平台

| 平台 | 字数 | 特点 |
|------|------|------|
| 朋友圈 | 50-150字 | 简洁有温度 |
| 小红书 | 200-500字 | 种草感强 |
| 电商详情 | 300-800字 | 转化导向 |
| 私域带货 | 100-300字 | 促单明确 |
| 抖音/快手 | 50-200字 | 前3秒抓人 |
| 微博 | 100-300字 | 话题性强 |
| B站 | 200-600字 | UP主口吻 |
| Instagram | 50-300字 | 英文+hashtag |

### 4. 选择品牌层级

| 层级 | 调性 |
|------|------|
| 大众品牌 | 口语化，性价比导向 |
| 中端品牌 | 有质感，生活方式导向 |
| 高端品牌 | 克制精炼，故事感导向 |
| 奢侈品牌 | 极简留白，意象化表达 |

### 5. 选择文案风格

- **温柔种草** - 细腻柔和，情感共鸣
- **简约高级** - 精炼克制，质感导向
- **接地气带货** - 直白亲切，促单明确

### 6. 选择输出语言

- 跟随平台默认（国内中文，海外英文）
- 中文 / 英文 / 中英双语

### 7. 获取文案

系统生成 2-3 个版本的文案供选择，每个版本附带【字数统计】【风格标签】，可直接使用或微调。

## 示例

**输入图片：** 奶白色陶瓷杯，金色描边，置于木质桌面

**识别结果：**
```
外观：圆润杯身，奶白色调
材质：陶瓷，哑光质感
细节：杯口金色描边
场景：木质桌面，温馨居家感
```

**输出文案（小红书/温柔种草）：**

```
标题：被问爆了的宝藏好物

真的被这个杯子狠狠拿捏了！

奶白色的陶瓷质感，拿在手里沉甸甸的很有分量。
杯口的金色描边是点睛之笔，精致又不张扬。
放在书桌上，搭配一束小花，整个角落都温柔起来了。

每天用它喝水，心情都会变好。
真心推荐给喜欢极简风的朋友们～
```

**输出文案（Instagram/英文）：**

```
Simple elegance for your everyday ☕
This cream-white ceramic mug with gold trim is everything.
Minimalist design, maximum vibes.

Shop via link in bio 🔗

#ceramicmug #minimalstyle #desksetup #aestheticfinds
```

## 核心原则

1. **只写真实存在的东西** - 不虚构产品特性
2. **说人话** - 不用"匠心打造""品质之选"等空洞词汇
3. **适配场景** - 不同平台、不同品牌层级用不同写法
4. **多版本可选** - 提供选择，而非唯一答案
5. **隐私保护** - 含人脸时提醒裁剪遮挡
