# 多图组合模式指南

本文件详细说明多图上传时的处理策略、主图选择规则和多图文案生成方法。

---

## 多图识别策略

| 图片数量 | 处理方式 |
|----------|----------|
| 1张 | 单图模式，标准流程 |
| 2-3张 | 组合模式：主图+细节图/对比图 |
| 4张及以上 | 图文模式：按场景/角度/对比分类识别 |

---

## 主图选择规则

多图模式下，按以下优先级确定主图：

| 优先级 | 选择标准 | 说明 |
|--------|----------|------|
| 1 | 用户明确指定 | 用户主动指出哪张是主图 |
| 2 | 清晰度最高 | 分辨率最高、主体最突出的图片 |
| 3 | 产品主体最大 | 产品在画面中占比最大的图片 |
| 4 | 第一张上传 | 默认选择用户上传的第一张图片 |

### 主图确认话术

```
检测到您上传了X张图片，我将以此图为主图进行描述：
[展示主图缩略图]

其他图片将用于补充细节和场景。
如需更换主图，请告诉我（如"用第2张做主图"）。
```

---

## 多图类型识别与处理

| 图片类型 | 识别特征 | 处理方式 | 文案比重 |
|----------|----------|----------|----------|
| **主图** | 产品完整、主体突出、光线最佳 | 作为主要描述依据 | 60% |
| **细节图** | 特写、局部放大、工艺展示 | 用于补充亮点描述 | 20% |
| **场景图** | 使用环境、搭配元素、氛围营造 | 用于场景描述 | 15% |
| **对比图** | 前后对比、大小对比、竞品对比 | 用于突出优势 | 5% |

### 图片类型识别要点

**主图识别：**
- 产品占据画面中心位置
- 光线充足，清晰可见
- 背景简洁，不干扰主体
- 展示产品完整形态

**细节图识别：**
- 特写镜头，局部放大
- 展示工艺、材质、纹理
- 可能有微距效果
- 突出某个特定部位

**场景图识别：**
- 展示产品使用环境
- 有搭配元素（家具、装饰等）
- 营造氛围和生活方式感
- 产品不一定是画面绝对中心

**对比图识别：**
- 包含两个或多个对象
- 展示大小、颜色、效果对比
- 可能包含使用前后的变化
- 突出产品优势

## 多图文案生成规则

1. **主图优先** - 按上述优先级确定主图，作为主要描述依据
2. **细节补充** - 细节图用于补充亮点、工艺、材质信息
3. **场景营造** - 场景图用于营造使用氛围和生活方式感
4. **对比突出** - 对比图用于强化产品优势和差异化
5. **逻辑串联** - 多图信息需自然串联，不罗列堆砌
6. **统一调性** - 多图文案保持一致的风格和语气

### 多图文案模板

```
[主图产品概述] + [细节图补充亮点] + [场景图营造氛围] + [对比图突出优势] + [总结推荐]
```

### 多图文案示例

**示例：陶瓷杯多图组合**

```
标题：这个杯子，从打开包装的那一刻就被圈粉了

第一眼看到这个奶白色的杯身，就觉得好温柔。
拿起来才发现，沉甸甸的很有分量，陶瓷的哑光质感摸着特别舒服。
（主图描述，占比60%）

凑近看，杯口的金色描边做得真精致，弧度刚好贴合嘴唇。
（细节图补充，占比20%）

放在木质桌面上，旁边摆一盆小绿植，整个角落的气氛都变了。
（场景图补充，占比15%）

和家里原来的杯子一比，这个质感真的好太多了。
（对比图突出，占比5%）

真心觉得，生活里这些小物件选对了，每天都会多一点小确幸。
（总结）
```

## 多图模式交互流程

### 步骤1：检测图片数量

```
检测到您上传了3张图片，进入多图组合模式。
```

### 步骤2：自动识别图片类型

```
我识别到以下图片类型：
- 图1：主图（产品完整展示）
- 图2：细节图（杯口特写）
- 图3：场景图（桌面摆放）

识别是否正确？如有误请告诉我（如"图2是场景图"）。
```

### 步骤3：确认主图

```
我将使用【图1】作为主图进行主要描述。
如需更换主图，请告诉我（如"用图2做主图"）。
```

### 步骤4：生成多图文案

按照多图文案模板生成文案，确保各部分内容比重合理。

## 特殊情况处理

### 图片类型无法识别

当无法确定图片类型时：

```
无法确定第2张图片的类型，请选择：
- 1. 细节图（特写、工艺展示）
- 2. 场景图（使用环境、搭配）
- 3. 对比图（前后对比、大小对比）
- 4. 另一张主图（多角度展示）
```

### 多张同类型图片

当用户上传多张同类型图片时（如3张都是场景图）：

```
检测到您上传了多张场景图，我将选择光线最佳的一张作为主要场景描述，其他作为补充。
```

### 图片内容冲突

当多张图片内容存在冲突时（如颜色不一致）：

```
检测到图片中产品颜色存在差异（图1为奶白色，图2为米白色），请以哪张为准？
```
