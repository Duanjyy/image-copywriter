# 文案风格与品牌层级指南

本文件详细说明3种文案风格、4级品牌层级，以及它们的兼容性矩阵。

---

## 三大文案风格对比

| 风格类型 | 语言特点 | 适用场景 | 代表性表达 |
|----------|----------|----------|------------|
| 温柔种草 | 细腻柔和，情感共鸣 | 小红书、朋友圈 | "真的好喜欢这个..." |
| 简约高级 | 精炼克制，质感导向 | 电商详情、品牌调性 | "极简设计，质感呈现" |
| 接地气带货 | 直白亲切，促单明确 | 私域带货、直播话术 | "这个真的超值..." |

---

## 风格创作心法

### 温柔种草风格

**核心特征：**
- 用"我"的视角分享体验
- 细节描写要有画面感
- 情感真挚，不刻意煽情
- 语言柔和，有温度

**创作要点：**
- 开篇用感受引入，不直接推销
- 用具体细节替代泛泛而谈
- 可适当加入使用心情
- 结尾自然收束，不刻意引导

**示例：**
> "拿到手就被这个质感惊艳到了，拿在手里沉甸甸的很有分量。奶白色的陶瓷质感，杯口的金色描边是点睛之笔，放在办公桌上每次喝水都是一种小确幸。"

### 简约高级风格

**核心特征：**
- 用词精准，避免冗余
- 强调设计感和品质
- 留白感，不说满
- 结构清晰，层次分明

**创作要点：**
- 开篇直接点明产品定位
- 用简洁语句描述核心特点
- 避免形容词堆砌
- 结尾点到为止

**示例：**
> "极简线条，哑光质感，于细节处见匠心。圆润杯身，金色描边勾勒精致轮廓。不喧哗，自有声。"

### 接地气带货风格

**核心特征：**
- 口语化表达，像朋友推荐
- 突出性价比和实用价值
- 适当紧迫感引导
- 直接明了，不绕弯子

**创作要点：**
- 开篇直接推荐
- 用具体卖点说话
- 可带价格或优惠信息
- 结尾明确促单引导

**示例：**
> "这个真的要推荐给你们！陶瓷杯质感超好，奶白色很百搭，金色描边超精致。我自己都囤了两个，现在活动价很划算，想要的赶紧冲！"

## 四级品牌层级

不同品牌层级决定文案的调性和表达方式，生成文案前需确认品牌定位：

| 品牌层级 | 语言特点 | 表达策略 | 代表性表达 |
|----------|----------|----------|------------|
| 大众品牌 | 口语化、接地气 | 性价比导向，突出实惠实用 | "这个价格真的划算""性价比超高" |
| 中端品牌 | 有质感、有温度 | 生活方式导向，强调品质与场景 | "让日常更有仪式感""生活的小确幸" |
| 高端品牌 | 克制精炼、有深度 | 故事感导向，强调理念与品位 | "于细节处见真章""少即是多" |
| 奢侈品牌 | 极简留白、意象化 | 意象化表达，强调稀缺与传承 | "不言自明""时光雕琢" |

### 各层级创作要点

#### 大众品牌

- 用最日常的语言，像朋友推荐
- 突出价格优势和实用价值
- 可直接点明"划算""超值""囤了几个"
- 避免过度包装，真诚直接

**示例：**
> "这个杯子真的便宜又好看，奶白色百搭，金色描边精致，我自己都囤了两个。"

#### 中端品牌

- 语言有温度但不夸张
- 强调生活品质和场景感
- 适度使用文艺表达，但不刻意
- 注重"性价比"和"品质感"的平衡

**示例：**
> "奶白色的陶瓷杯身，放在办公桌上，让每天的喝水时光都多了一份仪式感。"

#### 高端品牌

- 用词克制精炼，不堆砌形容词
- 强调设计理念和工艺细节
- 可融入品牌故事或设计灵感
- 留白感，点到为止

**示例：**
> "圆润杯身，哑光釉面，金色描边勾勒于杯口。不喧哗，自有声。"

#### 奢侈品牌

- 极简表达，惜字如金
- 意象化、诗意化，不直白推销
- 强调传承、稀缺、独特性
- 可留白处理，让读者自行感受

**示例：**
> "白瓷如玉，金线为骨。一杯一世界。"

## 品牌层级与风格兼容性矩阵

为避免品牌调性与文案风格冲突，建议遵循以下搭配原则：

| 品牌层级 | 推荐风格 | 可选风格 | 不建议风格 |
|----------|----------|----------|------------|
| 大众品牌 | 接地气带货 | 温柔种草 | 简约高级 |
| 中端品牌 | 温柔种草 | 简约高级、接地气带货 | - |
| 高端品牌 | 简约高级 | 温柔种草 | 接地气带货 |
| 奢侈品牌 | 简约高级 | - | 温柔种草、接地气带货 |

### 冲突提示话术

当用户选择冲突组合时，使用以下话术提示：

```
您选择了【奢侈品牌 + 接地气带货】的组合，这可能会削弱品牌调性。
建议改为：
- 奢侈品牌 + 简约高级（推荐）
- 或 中端品牌 + 接地气带货

是否继续当前组合？
```

### 兼容性检查逻辑

```
IF 品牌层级 = 奢侈品牌 AND 文案风格 = 接地气带货:
    提示冲突，推荐替代方案
ELSE IF 品牌层级 = 大众品牌 AND 文案风格 = 简约高级:
    提示可能过于高冷，建议确认
ELSE IF 品牌层级 = 高端品牌 AND 文案风格 = 接地气带货:
    提示可能削弱品牌调性
ELSE:
    正常生成文案
```

## 中英双语创作要点

### 英文文案创作要点

- **自然地道** - 使用母语者常用的表达方式，不生硬翻译
- **简洁有力** - 英文文案倾向于更短、更直接
- **善用 hashtag** - Instagram/Twitter 等平台 hashtag 是重要流量入口
- **CTA 明确** - 结尾引导清晰（Shop now / Link in bio / Tap to learn more）
- **避免中式英语** - 不直译中文表达习惯

### 中英对照示例

| 中文 | 英文 |
|------|------|
| 被这个质感圈粉了 | Obsessed with this texture |
| 放在桌上就是一道风景 | Looks like a piece of art on the desk |
| 每天用它心情都变好 | Makes my day every time I use it |
| 性价比真的高 | Such a steal / Great value for money |
| 赶紧冲 | Don't walk, run / Grab yours now |
| 极简设计 | Minimalist design |
| 质感绝了 | The texture is everything |
| 生活的小确幸 | Little joys in life |
| 仪式感 | Sense of ritual |
| 不喧哗，自有声 | Silent but powerful |
